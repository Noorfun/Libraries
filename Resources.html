Resources

Discover the wide range of resources available at Fairfax County Public Library, including books, digital media, research databases, and community programs that support diverse interests and needs.

George Mason Regional Library
This schools provides provides important resources that help FCPL function as a community. This library contains free computers that allow people to complete school work, research, and job applications. A key component of the library is the community food pantry that supports families by offering nonperishable food and encouraging donations.
Pine Spring Elementary School Library
This library is a resource hub for students. An example of this is the Destiny book catalog for both the books in the library but also E-books and databases for further research. These tools allow students to find reliable sources for projects and independent reading. By providing organized research materials, the library helps students study and access information more effectively.
Mantua Elementary School Library
The Mantua Elementary Library also uses the Destiny book catalog and has a large amount of digital access to online databases. Students can read, research, and study using both print and digital materials. These resources expand learning opportunities and allow students to access information beyond the classroom.
